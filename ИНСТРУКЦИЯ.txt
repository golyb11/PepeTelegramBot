================================================================================
                    ИНСТРУКЦИЯ ПО НАСТРОЙКЕ И ЗАПУСКУ БОТА "ПЭПЭ"
================================================================================

1. КАК ЗАПУСТИТЬ БОТА
--------------------------------------------------
В папке с ботом (pepe_telegram_bot) есть два удобных файла:
- start_bot.bat — двойной клик по этому файлу запустит бота. Откроется черное 
  окно консоли, где будут писаться логи. Бот работает, пока открыто это окно.
- stop_bot.bat — двойной клик по этому файлу принудительно остановит бота 
  (закроет окно с работающим ботом).

2. КАК УСТАНОВИТЬ АВАТАРКУ БОТУ (photo_2026-02-22_17-05-37.jpg)
--------------------------------------------------
Telegram API не позволяет ботам программно менять свою собственную аватарку 
через обычный код. Это сделано в целях безопасности.
Чтобы установить картинку (которая лежит в папке TgBot) на аватар:
  1. Откройте Telegram и найдите официального бота @BotFather.
  2. Напишите ему команду: /setuserpic
  3. Выберите вашего бота (появится кнопка в чате, например @PepedronerBot).
  4. BotFather попросит отправить ему фотографию. Отправьте ему файл 
     "photo_2026-02-22_17-05-37.jpg" из вашей папки TgBot.
  5. BotFather ответит "Success! Profile photo updated." — аватарка установлена!

3. КАК ПОЛЬЗОВАТЬСЯ БОТОМ В ГРУППЕ "ОГОРОД"
--------------------------------------------------
Бот имеет два режима работы:
  Режим А (Тролль "Пэпэ"):
  - Напишите в чате любое сообщение, содержащее слово "Пэпэ" (или "пэпэ").
  - Пример: "Пэпэ, что думаешь про погоду?"
  - Бот ответит жестко, саркастично и в своем токсичном стиле.

  Режим Б (Умный помощник):
  - Используйте команду /ask и ваш вопрос.
  - Пример: "/ask В каком году человек полетел в космос?"
  - Бот ответит вежливо, точно и подробно, без шуток.

4. СМЕНА API КЛЮЧА И МОДЕЛИ ПРЯМО В ЧАТЕ
--------------------------------------------------
Если бесплатная модель перестала работать или вы хотите сменить API ключ OpenRouter, 
любой участник группы может сделать это прямо в чате:
  - Смена ключа: /setkey ВАШ_НОВЫЙ_КЛЮЧ
  - Смена модели: /setmodel ИМЯ_НОВОЙ_МОДЕЛИ (например: /setmodel google/gemini-2.5-flash)

Все настройки автоматически сохраняются в базу данных (bot_settings.db) и не 
слетают при перезапуске бота!
